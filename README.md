# E-Commerce Analytics Dashboard

## Overview
End-to-end analytics pipeline processing 50K+ e-commerce transactions on Databricks with 6 professional visualizations.

## Dashboard Features
✅ KPI Summary (4 key metrics)
✅ Revenue by Region (Bar Chart)
✅ Daily Revenue Trend (Line Chart)
✅ Top 10 Products (Bar Chart)
✅ Category Distribution (Pie Chart)
✅ Customer Segments (Table)

## Architecture
- **Data Source:** 50,000 e-commerce transactions
- **Platform:** Databricks Community Edition
- **Storage:** Delta Lake (Bronze/Silver/Gold layers)
- **Processing:** Apache Spark + SQL
- **Visualization:** Databricks Dashboard

## Key Metrics
- Total Customers: 4,987
- Total Orders: 50,000
- Total Revenue: $1.2B+
- Avg Order Value: $25,543
- Return Rate: 2.45%

## Technologies Used
- Databricks
- Apache Spark
- Delta Lake
- SQL
- Python

## How to Replicate
1. Create Databricks Community Edition account
2. Create cluster (i3.xlarge, 1-2 workers)
3. Create 5 notebooks with provided code
4. Run queries in order
5. Create dashboard with 6 visualizations
